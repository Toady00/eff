This gems aims to be a way to help simplify the task of building multiple packages with [FPM](https://github.com/jordansissel/fpm).

## Installation

Add this line to your application's Gemfile:

    gem 'eff'

And then execute:

    $ bundle

Or install it yourself as:

    $ gem install eff
